# Requisitos No Funcionales
### Plataforma y Tecnología
El sistema será una aplicación web desarrollada con **Java** utilizando el framework **Spring**.
El sistema no requerirá la instalación de software adicional por parte del usuario más allá de un navegador web.

### Navegadores Compatibles
El sistema deberá funcionar correctamente en las versiones actuales de **Google Chrome**, **Mozilla Firefox** y **Microsoft Edge**.
No se garantiza compatibilidad con navegadores desactualizados o en desuso (como Internet Explorer).

### Sistemas Operativos
Al tratarse de una aplicación web, el sistema será compatible con cualquier sistema operativo que disponga de un navegador soportado (**Windows**, **Linux**, **macOS**).

### Dispositivos
El sistema estará orientado principalmente al uso desde **computadoras de escritorio y laptops** a **tablets y dispositivos móviles**.
Se intentará que la interfaz sea razonablemente usable desde dispositivos móviles, aunque no es un requerimiento prioritario para esta etapa del proyecto.

### Idioma
La interfaz del sistema estará en **idioma español**.
