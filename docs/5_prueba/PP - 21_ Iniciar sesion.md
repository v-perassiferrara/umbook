# PP - 21 : Procedimiento de Inicio de Sesión

Este procedimiento describe de manera genérica los pasos para probar el inicio de sesión y autenticación de usuarios.

### Pasos de Ejecución:
1. Dirigirse a la sección **"Iniciar Sesión"**.
2. Completar el campo **"Email"** con el correo electrónico correspondiente.
3. Completar el campo **"Contraseña"** con la contraseña correspondiente.
4. Presionar el botón **"Iniciar Sesión"**.
5. Verificar el comportamiento del sistema (redirección a la página de inicio, bloqueo de cuenta o mensaje de error de credenciales).
