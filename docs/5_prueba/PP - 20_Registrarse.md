# PP - 20 : Procedimiento de Registro de Usuario

Este procedimiento describe de manera genérica los pasos para probar el registro de un nuevo usuario en la plataforma.

### Pasos de Ejecución:
1. Dirigirse a la sección **"Registrarse"**.
2. Completar el campo **"Nombre"** con el nombre correspondiente.
3. Completar el campo **"Apellido"** con el apellido correspondiente.
4. Completar el campo **"Email"** con el correo electrónico correspondiente.
5. Completar el campo **"Nombre de usuario"** con el nombre de usuario correspondiente.
6. Completar el campo **"Contraseña"** con la contraseña correspondiente.
7. Seleccionar la **"Fecha de nacimiento"** correspondiente.
8. Presionar el botón **"Crear cuenta"**.
9. Verificar el resultado en la pantalla y el comportamiento de redirección, o en su defecto, la presencia y contenido de los mensajes de error/éxito en la interfaz.
