# PP - 1.3 : Procedimiento de Búsqueda de Usuarios

Este procedimiento describe de manera genérica los pasos para probar la funcionalidad de búsqueda de usuarios en la plataforma.

### Pasos de Ejecución:
1. Iniciar sesión en el sistema con un usuario válido.
2. Dirigirse a la sección **"Buscar Usuarios"**.
3. Completar el campo de búsqueda con el término o criterio correspondiente.
4. Presionar el botón **"Buscar"**.
5. Verificar el listado de resultados mostrados, o en su defecto, el mensaje de error o aviso en pantalla.
